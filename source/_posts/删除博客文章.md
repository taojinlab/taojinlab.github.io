# 删除博客文章

<!-- more -->

使用 Hexo 写博客时，如果以后想删除某一篇已经发布过的文章，不能只在网页上删除，因为 GitHub Pages 展示的是 Hexo 生成后的静态网页。

正确做法是：

```text
删除本地 Markdown 原文 -> 重新生成网页 -> 重新发布到 GitHub Pages
```

## 第一步 进入博客目录

打开 PowerShell，进入本地博客文件夹：

```powershell
cd "D:\code data\my-blog"
```

## 第二步 找到要删除的文章

博客文章一般都保存在：

```text
D:\code data\my-blog\source\_posts
```

可以在 PowerShell 中查看文章列表：

```powershell
dir "D:\code data\my-blog\source\_posts"
```

也可以用 VS Code 打开博客文件夹：

```powershell
code .
```

然后进入：

```text
source/_posts
```

找到要删除的 `.md` 文件。

例如，要删除：

```text
旧文章.md
```

对应文件就是：

```text
D:\code data\my-blog\source\_posts\旧文章.md
```

## 第三步 删除 Markdown 文件

可以直接在 VS Code 文件列表中删除这篇 `.md` 文件。

也可以使用 PowerShell 删除：

```powershell
Remove-Item "D:\code data\my-blog\source\_posts\旧文章.md"
```

注意：删除前要确认文件名正确，避免误删其他文章。

## 第四步 本地重新预览

删除文章后，先重新启动本地预览：

```powershell
npx hexo clean
npx hexo generate
npx hexo server
```

浏览器打开：

```text
http://localhost:4000/
```

检查被删除的文章是否已经从首页、归档页、标签页中消失。

确认没有问题后，在 PowerShell 中按：

```text
Ctrl + C
```

停止本地预览服务。

## 第五步 保存源码变化

删除文章后，需要把这个删除操作保存到 GitHub 的 `main` 分支。

执行：

```powershell
git add .
git commit -m "Remove old post"
git push
```

这一步保存的是博客源码变化，也就是：

```text
某一篇 Markdown 文章已经被删除
```

## 第六步 重新发布网站

保存源码后，需要重新发布网站：

```powershell
npx hexo clean
npx hexo generate
npx hexo deploy
```

这一步会重新生成静态网页，并把新的网页文件上传到 GitHub 仓库的 `gh-pages` 分支。

发布完成后，等待几十秒，打开博客：

```text
https://taojinlab.github.io/my-blog/
```

确认被删除的文章已经不再显示。

## 为什么删除文章后还要重新发布

Hexo 博客有两类文件：

```text
source/_posts：Markdown 原文
public：生成后的网页文件
```

删除 `source/_posts` 中的 Markdown 文件，只是删除了文章原文。

网站真正访问的是发布到 `gh-pages` 分支的静态网页。只有执行：

```powershell
npx hexo clean
npx hexo generate
npx hexo deploy
```

GitHub Pages 上的网页才会更新。

## 删除文章的常用流程

以后删除某篇文章，可以按下面流程操作：

```powershell
cd "D:\code data\my-blog"
```

删除文章文件，例如：

```powershell
Remove-Item "D:\code data\my-blog\source\_posts\旧文章.md"
```

本地检查：

```powershell
npx hexo clean
npx hexo generate
npx hexo server
```

确认没问题后停止预览，然后保存源码：

```powershell
git add .
git commit -m "Remove old post"
git push
```

最后发布网站：

```powershell
npx hexo clean
npx hexo generate
npx hexo deploy
```

简单记忆：

```text
删 Markdown 文件 -> 本地预览确认 -> git push 保存源码 -> hexo deploy 更新网站
```
